# ProyectoHashTableEDD

Se creo un programa que almacena investigaciones, permite la búsqueda de estas mismas por autor, palabras claves y titulos, genera un archivo para continuar con una sesión previa y muestra la frecuencia con la que cada palabra clave aparece en un resumen.
